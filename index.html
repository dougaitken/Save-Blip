Save Blip

Shell scripts for saving your Blipfoto journal to pdf files on your computer.

About

In March 2015, Polaroid | blipfoto entered into liquidation. As with any possible service shutdown, Blippers looked for help to keep their journals in the event of Polaroid Blipfoto shutting up shop.

This originally started as a shell script put together by Tim Read after this news. After a shell script and Windows Batch script, Tim wanted to go further. The original scripts simply took a screenshot of the journal page and saved this as a pdf, it didn't provide the original image. The next stage hopes to address this.

This is completely unrelated and not-endorsed by Blipfoto or PLR IP Holdings, LLC

Right now you can download the Windows batch file or the Shell script.
